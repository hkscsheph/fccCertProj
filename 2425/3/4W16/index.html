書院七大最佳食煙地方

作為一位書院學生,隨身帶包煙都好正常,但係每食一次煙都係一個踢出校危機。為此我哋整理咗喺學校最佳嘅食煙地區,以上嘅地區都可以減低被發現嘅機率,避免我哋親愛嘅學生被踢出校。

書院大門外

學校話唔俾食我就唔食咩?

學校大門是全書院最多人食煙的地方,該地方的灰色地磚是一個灰色地帶,不是政府管理,也不是學校範圍,使在該地食煙可以合法化。但由於該地是和外面的延文禮士道連接著的,會令校風受創,也會令街坊反感從而向教育局舉報。

入口

  • 延文禮士道30號
  • 書院門口

    特性

  • 灰色地帶
  • 開揚

    黑樓梯

    伸手不見五指...

    黑樓梯其獨特的結構,使其能永久的黑暗,也因為該特性,該樓梯使用率偏低,有絕對的隱閉性,但由通風結構不足,煙味難以擴散,下午時會有幾位中四同學經常經過此地,他們會把椅子推下樓梯,近年來加裝了天眼,令食煙者被發現的機率增加。

    入口

  • 地下書院叢林
  • 1樓整野房
  • 2樓小劇場隔離
    •

    特性

  • 黑暗
  • 少人
    •
    天台

    書院跳樓題材拍攝場地

    天台是書院國最大農田,教導初中同學可以吃什麼草是其最主要的作用。由於對書同學和mentor來說,天台沒有任何作用,因此人流比較少,但由於經常有在上面拍莫名其妙題材的影片,如奇怪的足球片或是在天井跳樓的片,在project week時特別明顯。如果被發現了,你可以告訴他自己在拍莫名其妙題材的影片,看誰阻止自己食煙。

    入口

  • 正樓梯第二高層
  • 東北面後樓梯最高層
  • 西南面後樓梯最高層
  • 升降機R層
  • 找某名metor攞遊繩從天井上去
    •

    特性

  • 開揚
  • 很多奇怪人
  • 偏遠

    書院叢林

    城市郊野公園

    書院叢林是位於書院後門的巨大綠化樹林區,那裹的樹長到2樓高,該地能塞下無數個班房,面積神似後室。由於有樹林保護和巨大的區域,使得在這裹食煙難以被發現,再加上除了有時有零聲的足球隊小伙下來叢林執波,其餘時間沒什麼人出現,令被發現機率更低。

    入口

  • 創意大道
  • 後門
  • 石屋爬牆(切入)
    •

    特性

  • 極大面積
  • 偏僻
  • 隱蔽

    書院（晚上）

    夜深人靜對面有鬼

    書院人類在夜晚7點後就會被趕走,在此時段校工會開始鎖門,因此在他鎖門之前躲在班房裹,就可以使班房成為絕對沒有人能進入的密室,再加上在昏暗的環境下會更難發現。在這種情況下食到一氧化碳中毒依舊沒人能發現。

    但畢境學校對面是一個墳場,所以沒人發現不代表沒鬼發現。

    入口

    特性

    好恐怖,好恐怖呀!

    陳享哥是一個書院裹的恐怖黑色生物,用一隻手指就可以做成毀天滅地的變化,根據陳享哥紀錄片,其擁有穿牆、改變時空、吞噬進化的能力，以及他比梵塔黑更黑，吸收光度高達99.99923018459665%。其已導致上十億人死亡＆上百億美金損失，官方指應留在家中被難。此等巨獸可使人類絕種在書院，因此即使吸煙也沒有人幹擾也沒有人會發現更沒有任何後果。
